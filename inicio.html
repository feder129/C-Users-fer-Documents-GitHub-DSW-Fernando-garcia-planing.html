objetivos generales

El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. por ejemplo, sus contenidos podria ser parrafos, una lista de viñetas, o imagenes y tabla de datos. aqui nos enfocaremos en estos los codigos semanticos ( SECTION, ASIDE, FOOTER, ARTICLE) que se pueden utilizar para definir diferentes partes de una pagina web.
